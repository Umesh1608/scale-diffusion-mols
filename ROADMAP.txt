1.1  Implement E(n)-Equivariant GNN (EGNN) Blocks
    [ ] Implement EGNN message passing layer:
        m_{ij} = phi_e(h_i, h_j, ||x_i - x_j||^2, a_{ij})
    [ ] Implement coordinate update:
        x_i' = x_i + sum_j (x_i - x_j) phi_x(m_{ij})
    [ ] Implement feature update:
        h_i' = phi_h(h_i, sum_j m_{ij})
    [ ] Implement edge feature handling (bond type, distance)
    [ ] Implement multi-head attention variant of EGNN
    [ ] Implement skip connections and layer normalization
    [ ] Stack L EGNN layers (L=4-8)
    [ ] Verify E(n) equivariance with unit tests on random inputs
    [ ] Benchmark forward/backward pass time and memory

1.2  Implement Molecular Flexi-Net Architecture
    [ ] Implement hierarchical EGNN with multi-resolution processing:
        - Fine level: atom-level graph
        - Coarse level(s): coarsened graph from spectral clustering
    [ ] Implement dynamic activation functions (learnable per-layer)
    [ ] Implement inter-resolution message passing:
        - Fine-to-coarse: aggregate via C^{(k)}
        - Coarse-to-fine: distribute via C^{(k)T}
    [ ] Implement time embedding (sinusoidal + MLP)
    [ ] Implement resolution-level embedding
    [ ] Implement atom type embedding (one-hot or learned)
    [ ] Implement noise level conditioning (inject sigma_t into network)
    [ ] Implement output heads:
        - Epsilon prediction (noise) for coordinates
        - Epsilon prediction for atom features
    [ ] Model parameter count target: ~10-50M parameters
    [ ] Verify full model equivariance end-to-end

1.3  Implement Forward Diffusion Process with Spectral Coarsening
    [ ] Implement forward process: q(z_t | z_0) = N(z_t; M_t z_0, sigma_t^2 I)
    [ ] Implement resolution schedule r(t) with configurable breakpoints
    [ ] Implement coordinate diffusion (3D positions)
    [ ] Implement feature diffusion (atom types as continuous then discretized)
    [ ] Implement joint coordinate + feature diffusion
    [ ] Handle variable-size molecules in batched forward process
    [ ] Verify forward process marginals match analytic form
    [ ] Visualize forward process trajectories for sample molecules

1.4  Implement Reverse Process with Non-Isotropic Posterior
    [ ] Implement posterior mean: mu_{t-1|t,0} using M_t, M_t^T
    [ ] Implement posterior covariance: Sigma_{t-1|t,0} (non-isotropic)
    [ ] Implement DDPM-style reverse step:
        z_{t-1} = mu_{t-1|t,0}(z_t, epsilon_theta) + Sigma^{1/2} * noise
    [ ] Implement DDIM-style deterministic reverse step (optional)
    [ ] Implement ancestral sampling with full covariance
    [ ] Implement Lanczos-accelerated posterior sampling
    [ ] Implement number-of-atoms prediction/conditioning
    [ ] Handle resolution transitions cleanly during reverse process
    [ ] Verify reverse process produces valid molecular geometries

1.5  Implement Noise Schedules
    [ ] Implement ConvexDecay schedule:
        alpha_t = 1 - (t/T)^p for convexity parameter p
    [ ] Implement SigmoidLikeDecay schedule:
        alpha_t = sigmoid(-a*(t/T - b)) / sigmoid(a*b)
    [ ] Implement linear schedule (baseline)
    [ ] Implement cosine schedule (baseline, from Nichol & Dhariwal)
    [ ] Implement schedule interpolation and blending
    [ ] Plot SNR curves for all schedules
    [ ] Make schedule selection configurable via hydra config

1.6  Implement Min-SNR-5 Loss Weighting
    [ ] Implement SNR calculation: SNR(t) = alpha_t^2 / sigma_t^2
    [ ] Implement Min-SNR-gamma weighting:
        w(t) = min(SNR(t), gamma) / SNR(t), gamma=5
    [ ] Implement weighted MSE loss on epsilon prediction
    [ ] Implement VLB loss term (optional, for hybrid objective)
    [ ] Compare training curves with and without Min-SNR weighting
    [ ] Validate loss weighting reduces variance of gradient estimates

6.1  Synthetic Route Planning
    [ ] Run retrosynthetic analysis on top 10-20 candidates:

        - Manual expert review of proposed routes
    [ ] Identify commercially available starting materials
    [ ] Estimate synthesis complexity (number of steps, expected yield)

6.2  Synthesis and Characterization
    [ ] Synthesize top 10-20 candidates
    [ ] Purification (column chromatography, recrystallization)
    [ ] Characterization by NMR:
        - 1H NMR (confirm structure, purity)
        - 13C NMR (confirm carbon skeleton)
        - 2D NMR if needed (COSY, HSQC, HMBC)
    [ ] Mass spectrometry (ESI-MS or MALDI-TOF, confirm molecular weight)
    [ ] IR spectroscopy (confirm functional groups)
    [ ] Melting point determination
    [ ] Purity assessment (HPLC, target >95%)
    [ ] Report synthesis yield and any difficulties encountered

6.3  UV-Vis Absorption Spectroscopy
    [ ] Measure UV-Vis absorption spectra in solution:
        - Solvents: ethanol, cyclohexane, and coating-relevant solvent
        - Concentration series for epsilon determination
        - Wavelength range: 200-800nm
    [ ] Determine lambda_max, epsilon_max, bandwidth
    [ ] Compare experimental spectra with:
        - Model-predicted spectra
        - TD-DFT-predicted spectra
    [ ] Compute prediction errors (MAE of lambda_max, spectral overlap)
    [ ] Assess broad-spectrum coverage (280-400nm)
    [ ] Measure spectra in solid state / thin film (if relevant)

6.4  Photostability Testing
    [ ] Prepare solutions and thin films of each candidate
    [ ] Xenon arc lamp exposure (simulating sunlight):
        - 500 hours continuous exposure
        - Monitor UV-Vis spectrum at 0, 50, 100, 200, 500 hours
    [ ] Calculate photostability:
        - % retention of absorbance at lambda_max
        - Rate of photodegradation
    [ ] Identify photodegradation products (LC-MS if needed)
    [ ] Compare with commercial benchmarks:
        - Tinuvin 326 (benzotriazole)
        - Chimassorb 81 (benzophenone)
        - Tinuvin 1577 (hydroxyphenyl triazine)
    [ ] Classify: photostable (>90% retention at 500h) vs unstable

6.5  Thermal Analysis
    [ ] TGA (Thermogravimetric Analysis):
        - Heating rate: 10 C/min, N2 atmosphere
        - Determine T_onset (decomposition onset), T_50% (50% weight loss)
        - Target: T_onset > 250 C for coating applications
    [ ] DSC (Differential Scanning Calorimetry):
        - Determine T_m (melting point), T_g (glass transition if amorphous)
        - Assess thermal transitions relevant to processing
    [ ] Thermal stability ranking of candidates

6.6  Coating Formulation and Testing
    [ ] Prepare coating formulations:
        - Epoxy matrix: mix UV absorber at 1-5 wt% in epoxy resin
        - Polyurethane (PU) matrix: mix at 1-5 wt% in PU coating
        - Control coatings (no UV absorber, commercial UV absorber)
    [ ] Cast/spray coatings on standard substrates (aluminum, steel, wood)
    [ ] Cure coatings per standard protocols
    [ ] Assess UV absorber compatibility:
        - Visual clarity (haze, yellowing)
        - Homogeneity (no phase separation or blooming)

6.7  UV Transmission and Accelerated Weathering
    [ ] Measure UV-Vis transmission of coated substrates:
        - Before weathering (baseline)
        - Target: <5% transmission at 280-380nm
    [ ] QUV accelerated weathering test:
        - UVA-340 lamps, 0.89 W/m2 at 340nm
        - Alternating cycles: 8h UV at 60C / 4h condensation at 50C
        - Duration: 1000-2000 hours
    [ ] Measure at intervals (0, 250, 500, 1000, 2000 hours):
        - UV transmission
        - Color change (Delta E, ASTM D2244)
        - Gloss retention (ASTM D523)
        - Visual inspection (cracking, chalking, delamination)
    [ ] Compare with commercial UV absorber coatings

6.8  Adhesion and Mechanical Properties
    [ ] Cross-cut adhesion test (ASTM D3359) before and after weathering
    [ ] Pull-off adhesion test (ASTM D4541) if relevant
    [ ] Pencil hardness (ASTM D3363)
    [ ] Impact resistance (ASTM D2794) if relevant
    [ ] Verify UV absorber does not compromise coating mechanical integrity

        UV protection (QUV)     |                   |             |               |
        Color stability (dE)    |                   |             |               |
